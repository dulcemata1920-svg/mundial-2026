## ⚽ App Mundial 2026

Una aplicación interactiva para seguir el Mundial de Fútbol 2026. Consulta la cuenta regresiva, partidos, tablas de posiciones y más.

### 🎯 Características

- **📅 Cuenta Regresiva**: Tiempo real hasta el inicio del torneo
- **🎯 Partidos**: Información detallada de todos los encuentros por fase
- **🏆 Posiciones**: Tablas de grupos con estadísticas completas
- **📱 Responsive**: Funciona perfectamente en dispositivos móviles
- **🎨 Interfaz Moderna**: Diseño atractivo con gradientes y animaciones

### 🚀 Inicio Rápido

1. **Clonar el repositorio**
```bash
git clone https://github.com/tu-usuario/mundial-2026.git
cd mundial-2026
```

2. **Instalar dependencias**
```bash
npm install
```

3. **Iniciar la aplicación**
```bash
npm start
```

La aplicación se abrirá en [http://localhost:3000](http://localhost:3000)

### 📦 Dependencias

- React 18.2.0
- React DOM 18.2.0
- Axios 1.4.0

### 📱 Secciones

#### Cuenta Regresiva
- Tiempo real hasta el 12 de junio de 2026
- Información sobre sedes, fechas y participantes
- Se actualiza cada segundo

#### Partidos
- Información de todos los encuentros
- Filtrado por fase (Grupos, Octavos, Cuartos, Semis, Final)
- Detalles de equipos, fechas y horarios

#### Posiciones
- Tablas de todos los grupos
- Estadísticas completas (PJ, G, E, P, GF, GC, DG, Pts)
- Clasificación en tiempo real
